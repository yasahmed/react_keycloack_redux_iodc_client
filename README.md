# React + Redux Boilerplate

## Building and running on localhost

First install dependencies:

```sh
yarn
```

To run in hot module reloading mode:

```sh
yarn start
```

To create a production build:

```sh
yarn build:prod
```

To create a development build:

```sh
yarn build:dev
```

## Testing

```sh
yarn test
```

To test in watch mode:

```sh
yarn test:watch
```

Coverage:

```sh
yarn test:coverage
```
